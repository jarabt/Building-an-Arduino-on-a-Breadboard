# Building-an-Arduino-on-a-Breadboard

Basic Parts for wiring up Arduino

    Breadboard
    22 AWG wire
    7805 Voltage regulator
    2 LEDs
    2 220 Ohm resistors
    1 10k Ohm resistor
    2 10 uF capacitors
    16 MHz clock crystal
    2 22 pF capacitors
    small momentary normally open ("off") button, i.e. Omron type B3F
